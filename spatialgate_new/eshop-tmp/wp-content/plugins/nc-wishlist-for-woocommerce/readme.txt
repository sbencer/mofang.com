This plugin offer your store visitors  to add the products of your store to a wishlist page.

== Description ==
This plugin offer your store visitors  to add the products of your store to a wishlist page.All the products added to the wishlist area allowed have displayed image,add to cart button,stock and price.

This plugin also allow you to restrict the use of the wishlist page and buttons to logged in users or all users.

== Installation Guide ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the setting link from plugin page.
